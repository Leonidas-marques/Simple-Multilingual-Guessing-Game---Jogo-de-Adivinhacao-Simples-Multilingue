
# Jogo de Adivinhação Simples Multilíngue

Feito em python com suporte multilíngue.


## Funcionalidades

- Auto-Tradução 
- 3 níveis de dificuldade
- Pontuação proporcional


## Dependências

Apenas uma biblioteca é necessária

```bash
  pip install deep_translator
```
